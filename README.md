
# GÉLICA Inventario (React + Vite + TypeScript)

## Requisitos
- Node.js 18+ y npm

## Pasos
1) Instalar dependencias:
   ```bash
   npm install
   ```
2) Ejecutar en desarrollo:
   ```bash
   npm run dev
   ```
   Abrí el enlace que aparece (por defecto http://localhost:5173).
3) Compilar para producción:
   ```bash
   npm run build && npm run preview
   ```
